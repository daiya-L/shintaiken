函館発新体験開発プロジェクトとは

私たちは様々なイベントを通して、私たちなりの「新体験」を発信し、伝えることを目指して活動しています。私たちは「新体験」を、「あたりまえに意識を向けて感性を豊かにする体験」と定義しました。この私達なりの「新体験」をもとにそれを伝えるコンテンツを開発しています。

私たちの活動

私たちは新体験の開発だけでなく、開発したコンテンツを様々な場所に出展しています。フィードバックから修正を繰り返し、より良いコンテンツにすることを目指しています。

2019.01.13

「新日常展」開催のお知らせ

一年間の活動の集大成として、函館市芸術ホールと未来大ミュージアムにて「新日常展」と題した展覧会を行います。さまざまな展示を通し、本プロジェクトの目的である「新体験の提供」が実現されることを目指しています。
